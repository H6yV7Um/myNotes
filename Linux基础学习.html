# Linux基础学习 ### Linux 界面 有 GUI（Graphical User Interface) 图形用户界面 和 CLI (command-line interface) 命令行界面 GUI 有 KDE 和 GNome CLI 有 bash, zsh, sh, tcsh, csh, ksh ### 远程连接 Linux 支持远程连接, 以ssh协议进行连接 ssh (secure shell) `` 1. 查看系统是否监听tcp协议的22端口 ss -tnl 2. 查看当前Linux机器的ip (inet addr后面的ip即机器的ip) ipconfig 3. 使用ping命令测试下远程机器是否能连接上 ping 172.26.182.23 4. 确保防火墙处于关闭状态. iptables -L -n 可以查看防火墙状态, 如果防火墙开启,需要关闭防火墙 systemctl disable firewalld.service // 停用防火墙, 适用centos7 systemctl stop firewalld.service // 关闭防火墙, 适用centos7 service iptables stop //停用防火墙, 适用 centos6 chkconfig iptables off //关闭防火墙, 适用 centos6 5. 用ssh 远程登录主机 ssh 172.26.182.23 ``` ![ss-tnl](images/chatu/ss-tnl.png) ![ip](images/chatu/ipconfig.png) echo $SHELL 查看当前所使用的shell版本.
