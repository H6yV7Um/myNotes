<h1>jQuery常用API</h1> ## I、选择器 ## II、核心API ## III、 Ajax请求 ### 1. <span class="red">$.ajax( ) 发送jsonp请求</span> ### 语法: $.ajax([settings]) url 是要访问的文件地址. [settings]里面是一些配置参数,用json来表示 ``` $.ajax({ "url":url, //jsonp文件地址 "dataType":"jsonp", //访问的数据类型 "jsonpCallback":callbackname, //设置jsonp中的函数名 "success":callback //请求完成后要做的事情. }) ``` ### 2. <span class="red">$.post( )发送post请求</span> ### 语法: $.post(url,data,fn,type) url : 必须,请求的地址 data : 必须,这次post请求发送的数据 fn : 必须,请求成功后的回调函数 type : 可选,返回数据的类型,可以是string或json ## IV、 属性 ## V、 文档处理 ## VI、 筛选 ## VII、 事件 ## VIII、 事件对象 ## IX、效果 ## X、工具 ## XI、 其他API ### 1. $.contains 包含 $.contains( dom1 , dom2 ) 判断一个dom1 是否包含dom2; 2个参数都是dom节点. 不是jQuery dom对象 ``` $.contains(document.documentElement, document.body); // true $.contains(document.body, document.documentElement); // false ``` ### 2. .data( key, value ) 数据缓存 在匹配元素上存储任意相关数据 或 返回匹配的元素集合中的第一个元素的给定名称的数据存储的值。 .data() 方法允许我们在DOM元素上绑定任意类型的数据,避免了循环引用的内存泄漏风险。 >* ### .data( key, value ) #### key 类型: String 一个字符串，用户存储数据的名称。（译者注：存储的数据名） >* ### .data( obj ) obj 类型: Object 一个用于更新数据的 键/值对 ``` $("body").data("foo", 52); //在jquery对象 的body元素上绑定数据(foo) $("body").data("bar", { myType: "test", count: 40 }); $("body").data({ baz: [ 1, 2, 3 ] }); $("body").data("foo"); // 52 //获取数据 $("body").data(); // { foo: 52, bar: { myType: "test", count: 40 }, baz: [ 1, 2, 3 ] } ```
