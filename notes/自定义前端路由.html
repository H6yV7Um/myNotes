# 自定义前端路由 适用于SPA 单页面应用程序中 原理, 根据当前页面的hash值的不同, 来显示不同的模块. 1. 监听页面的hash改变事件 ``` window.addEventListener("hashchange", route); ``` 2. 获得hash ``` var hash = location.hash; ``` 3. 处理hash并显示相应的组件 ``` <script type="text/javascript"> //路由 //要匹配的hash规则 http://localhost:3000/#list/search/fagad // list 表示要渲染的视图 后面的为其他功能 var route = function () { // 每次当hash改变的时候，我们要将它获取出来，根据hash值来渲染页面 //获得当前的hash var hash = location.hash; // 原hash类型 #\list\type\1 或者 #list\type\1 => 要得到的结果 list\type\1 hash = hash.slice(1).replace(/^\//,""); //拆分为数组 hash = hash.split("/"); // 以下是vue组件开发中选择视图的内容 // 根据hash值选择视图组件 app.view = hash[0]||"home"; // 父组件向子组件发送消息 成功通过父组件app向子组件发送消息 app.query = hash.slice(1); // $broadcast 父组件用来向子组件发送信息 app.$broadcast("reload-list") } // 监听页面的hash改变事件 window.addEventListener("hashchange", route); // 当首次打开页面的时候也进入route路由中 window.addEventListener("load", route); </script> ```
