1. 对象中属性名的特殊情况

以下4种情况对象中的k也必须加引号

  1. k以数字开头
  2. k是特殊符号
  3. k有空格
  4. k是关键字,保留字

辅助记忆: 叔父恐官 (数字,特殊符合,空格,关键字/保留字)

2. 对象的属性访问有两种方法

  1. 点语法 obj.k

    访问的属性名不用加引号

    点语法的缺点:1. 不能访问特殊格式的属性; 2. 不能访问变量属性名

  2. 方括号obj["k"]

    属性需要加引号, 变量类型的属性不加引号

    优势:1.可以访问特殊格式的属性; 2.可以访问变量属性

/*用new调用构造函数创建实例相当于隐秘执行了

function People(name,age){
    obj = {};
    obj.name = name;
    obj.age = age;
    obj.sayHello = function(){
        alert("hello");
    }
    return obj;
}
xiaoming = People("xiaoming",18);
console.log(xiaoming.name);

2 构造函数中的注意事项

构造函数中不能有return语句, 如果出现return语句会覆盖掉返回的对象,这样新生成的对象实例无法调用构造函数的属性和方法.

构造函数中如果没有this语句, 这个构造函数将不能为新创建的实例添加属性和方法,那这个构造函数也就没有意义了

构造函数名称的首字母一般大写,如function People(){ }
