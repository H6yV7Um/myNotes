I css3的选择器

1. 关系选择符

css3中新增的关系选择符有:

> 只能选择儿子,不能选择其他后代;
+ 选择一个兄弟, next sibling;
~ 选择所有的兄弟, next all siblings;
注意:用关系选择符的时候不要加空格,虽然加空格也不错,但是css中的空格有后代的意思,所以容易误解;

1.1 儿子选择符 " >"

.box>p

表示选择类名为box的元素的儿子p

Ie7开始兼容

1.2 下一个兄弟选择" + "

h3+p{ color : red ;}

选择h3标签的下一个兄弟元素p, 并且这个p需要是h3的第一个兄弟

< h3 > < /h3 >
<div > < /div >
< p > </p >
//这种情况, h3+p{ } 不能选择到p. // h3+div{ } 可以选择到div

1.3 选择后面所有兄弟"~" next all siblings

h3~p

表示选择页面上所有的 h3元素后面的p, 可以跨选择器

结果:

p2

h 2

div

p4

p5

注意: 在jQuery中 也可以使用以上的选择符, 没有兼容性问题

2. 属性选择器

css2.1中, 只能通过id, class 属性来选择元素;在css3中可以通过任意属性来选择元素了

2.1 语法: 属性选择器用方括号 [ ]

2.2 css3中选择器的扩展

2.2.1 ^表示以val开头

p[class ^= "para"]{ }

表示所有类名中包含以para开头的类名的元素

2.2.2 $ 表示以val结尾的

p[class $= "para"]{ }

表示所有类名中包含以para结尾的类名的元素

2.2.3 *表示属性值中包含val,可以出现在属性值中的任意部位

p[class *= "para"]{ }

表示类名值中包含para, 可以出现在类名中的任意部分

2.2.4 | 表示以val开头,后面紧跟一个"-"

p[class |= "para"]{ }

可以选择类名为 "para-haha" "para-" 之类的元素

2.2.5 ~ 表示选择val是独立词汇的元素

p[class ~= "para"]

等价于 p.para{ }
表示选择类名中含有para的元素p

结果:

1

2

3

4

5

3. 儿子选择器/序号选择器

这类选择器用" : "冒号来选择一个元素的儿子元素

3.1 first-child, last-child

p : last-child

只要一个p是页面上某元素的最后一个儿子就选择它
会选择页面上所有的是最后一个儿子的p

3.2 儿子选择器通过序号选择 ( nth-child( 1 ) )

3.2.1 p:nth-child(1)

p是某个元素的第一个子元素的话就选择它

3.2.2 p:nth-child(3n+8)

只要一个p是某一个元素的第8、11、14、17……个儿子，那么选择它

3.2.3 p:nth-last-child(3)

只要一个p是某个元素的倒数第3个儿子，那么选择它

3.2.4 .haha:nth-of-type(2)

只要一个元素有haha类并且是同种标签的第2个儿子，那么选择它

nth-of-type 表示同类型标签
