js及jQuery中各种尺寸的获取与设置

1.获取元素相关的尺寸

1.1 获取当前的宽高

1.1.1 jquery中

获取的是内容物的宽高, 不含边框border, 不含padding

$("div").width() 获取div的宽
$("div").width(100)设置div的宽为100px
$("div").height() 获取div的宽
$("div").height(100)设置div的宽为100px

获取的是内容物的宽高, 含边框border, 含padding

$div.outerHeight()
获取div的高包含外边框 $div.outerWidth()

1.1.2 js中

oDiv.offsetWidth
oDiv.offsetHeight
获取当前元素的宽和高,全线兼容
该属性获得的width height 是自己的width + 左右padding +左右border的值.

1.2 获取当前的宽高(不含border)

1.2.1 js中

oDiv.clientWidth
oDiv.clientWidth
clientWidth就是自己的width+padding的值。 也就是说，比offsetWidth少了border。

如果盒子没有宽度，那么那么所有浏览器都将把px值当做clientWidth，而不是100% 如果盒子没有高度，用文字撑的，IE6 clientHeight是0，其他浏览器都是数值。

1.2.2 jQuery中

$div.innerHeight()
$div.innerWidth()
获取元素的宽高, 不含边框

1.3 获取元素到定位父盒子的距离

1.3.1 jquery中

$(选择器).offset() 返回 json,同时有 left 和top 值
$(选择器).offset().left/top 返回具体的参数

设置元素的偏移值
$(选择器).offset(val) val是json

1.3.2 js中

oDiv.offsetLeft
oDiv.offsetTop
获取元素到定位基准父盒子(offsetParent)的left,top值.

自定位, 父无边时该属性没有兼容性问题

1.4 获取css中的样式

1.4.1 jquery中

1.4.2 js中

1.4.2.1获取行内样式

oDiv.style.width
oDiv.style.height
通过获取css样式的方法来获取当前的宽高. 只能获取写在行内样式中的宽高

1.4.2.2 通过获取计算后样式来获取元素的宽高

window.getComputedStyle(oDiv).getPropertyValue("width") 获取计算后的样式的完整写法

var w = getComputedStyle(oDiv)["width"] 获取计算后的样式的简化写法

获取计算后样式,可以获得内嵌和外链样式表中的样式需要用获取计算后的样式这种方法.
var w = getComputedStyle(oDiv)["width"]
var h = getComputedStyle(oDiv)["height"]

IE6,7,8中

oDiv.currentStyle["width"]
oDiv.currentStyle["height"]
